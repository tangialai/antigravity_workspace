# Vision: Landing Page / Website (Odoo Edition)

**Description:** A marketing or informational website built using Odoo Website Builder.

## Core Features

- [ ] CMS (Content Management System)
- [ ] Contact Form (Lead Generation)
- [ ] SEO Optimization
- [ ] Mobile Responsiveness

## Architecture Recommendation

- **Backend:** Odoo (Python)
- **Frontend:** Odoo Website Theme (QWeb + Bootstrap)
- **Database:** PostgreSQL

## Typical Odoo Modules

- `website`
- `website_crm` (for Contact Form)
- `website_blog`
- `theme_default`
